Sugerencias y consejos para la plantilla Launchpad de Classic Express

La plantilla Launchpad de Classic Express ofrece un punto de partida para crear un launchpad de configuración que puede utilizarse con los productos de la marca Express.

Esta plantilla contiene código que puede utilizarse para iniciar el programa de instalación de un producto, visualizar documentación y suministrar contenido traducido.

Elementos a tener en cuenta acerca de la plantilla Launchpad de Classic Express:

  • Todos los mandatos y programas utilizados se definen en launchpadContent/global.properties.
  • Las aplicaciones de ejemplo se incluyen en DISK1 de la página Diseño de disco. Para añadir programas de instalación o aplicaciones personalizadas, defina la aplicación en launchpadContent\global.properties (siguiendo los ejemplos de las aplicaciones de ejemplo). Importe las aplicaciones en el disco adecuado del proyecto utilizando los controles de la pestaña Diseño de disco.
  • Todos los archivos HTML utilizan la misma hoja de estilo (content.css) que reside en el directorio launchpadContent.

La plantilla Launchpad de Classic Express contiene las siguientes páginas:

  • Bienvenida - La página de bienvenida ofrece generalmente una visión general del producto y una descripción de las características del launchpad. La página de bienvenida debe ser concisa, pero debe ofrecer al usuario opciones para buscar más información acerca del producto mediante un enlace al sitio Web del mismo, que se visualizará en una nueva ventana del navegador. La página de bienvenida contiene código para abrir un sitio Web externo en un navegador. Utilice la función viewPage() para visualizar ventanas nuevas en lugar de la función window.open() de JavaScript, ya que window.open() tiene más posibilidades de quedar bloqueada por un bloqueador emergente.
  • Información de release - La página de información de release proporciona generalmente información específica de un determinado release de un producto. Por ejemplo, instrucciones de instalación, cambios en los requisitos previos, principales características o arreglos aplicados en este release y contenido similar son informaciones adecuadas para una página de información de release. La página de información de release ofrece código que permite ver un PDF y ejecutar una aplicación.
  • Información de requisitos previos - La página de requisitos previos proporciona generalmente un control para la ejecución de un programa de verificación de requisitos previos que determina si los requisitos previos existen en el sistema y devuelve una lista de los requisitos previos que no ha detectado. La página de información de requisitos previos ofrece código que permite ver un PDF y ejecutar una aplicación.
  • Escenario de instalación - El contenido de escenarios de instalación proporciona generalmente los diversos procedimientos de instalación de un producto. Los usuarios pueden elegir un escenario de instalación específico en función de cómo tengan previsto utilizar el producto. Este contenido describe las diferencias entre los diversos escenarios de instalación que el usuario puede elegir y suministra información acerca de la razón por la que el usuario elegirá un escenario de instalación en lugar de otro. No todos los productos presentan escenarios de instalación. Si personaliza la plantilla Launchpad de Classic Express para un producto de este tipo, puede eliminar esta sección de la navegación. La página de escenarios de instalación ofrece código que permite ejecutar un programa personalizado con un argumento.
  • Información de migración - El contenido de migración describe generalmente un recorrido de migración que el usuario debe seguir para realizar la actualización desde una versión anterior de un producto a una más reciente. Si la información de migración no es relevante para un producto determinado, este contenido debe eliminarse de la navegación. La página de información de migración ofrece código que permite ver un PDF y ejecutar una aplicación.
  • Instalar - La página de instalación del producto ofrece generalmente un control para iniciar la instalación de un producto. Puede añadir varias páginas de instalación de producto a un launchpad de configuración para dar soporte a instalaciones de varios productos. La página de instalación ofrece código para iniciar la instalación de un producto.
  • Salir - El enlace de salida del menú de navegación permite salir del Launchpad.
