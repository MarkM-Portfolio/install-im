Classic Express ランチパッド・テンプレートのヒント

Classic Express ランチパッド・テンプレートは、Express 製品に使用できるセットアップ・ランチパッド作成のための開始点として準備されています。

このテンプレートには、製品のインストール・プログラムの開始、資料の表示、 およびローカライズされたコンテンツの提供のために使用できるコードが含まれています。

Classic Express ランチパッド・テンプレートに関する注意点:

  • 使用するすべてのコマンドおよびプログラムは、launchpadContent/global.properties に定義されています。
  • サンプル・アプリケーションは、「ディスク・レイアウト」ページの DISK1 に入っています。 カスタムのインストール・プログラムやアプリケーションを追加するには、launchpadContent\global.properties に アプリケーションを定義します (サンプル・アプリケーションの例に従ってください)。「ディスク・レイアウト」タブにあるコントロールを使用して、 プロジェクト内の適切なディスクにアプリケーションをインポートします。
  • すべての HTML ファイルは、launchpadContent ディレクトリーにある同じスタイルシート (content.css) を使用します。

Classic Express ランチパッド・テンプレートには次のページを含みます。

  • ようこそ -「ようこそ」ページには、通常、製品の概要と、ランチパッドの機能の説明が表示されます。 「ようこそ」のページは簡潔にまとめられていますが、ページ内にある製品 Web サイトへのリンクを ご利用になれば、製品の詳細情報を参照することができます。 「ようこそ」ページでは、ブラウザーに外部の Web サイトを表示させるためのコードが用意されています。 JavaScript の window.open() の代わりに、viewPage() 機能を 使用して新規ウィンドウを表示します。window.open() の方がポップアップ・ブロッカーでブロックされる可能性が高いためです。
  • リリース情報 - リリース情報のページでは通常、製品の固有のリリースに特定の情報が表示されます。例えば、 インストールに関する考慮事項、前提条件の変更、そのリリースに加えられた主要なフィーチャーまたは修正などの内容が、 リリース情報のページに適しています。リリース情報のページには、PDF の表示およびアプリケーションの実行のためのコードも用意されています。
  • 前提条件 - 前提条件のページでは通常、前提条件検証プログラムを実行することができます。 これにより、システムの前提条件がコンピューターで正しく準備されているかどうかを判別し、検出できなかった前提条件があれば そのリストを返します。前提条件のページには、PDF の表示およびアプリケーションの実行のためのコードも用意されています。
  • インストール・シナリオ - インストール・シナリオの内容には通常、 ユーザーが製品のインストールに選択できるシナリオがいくつか用意されています。 ユーザーは製品の使用目的や使用方法に基づいて、特定のインストール・シナリオを選択できます。このコンテンツでは、 ユーザーが選択できる各種のインストール・シナリオについてそれぞれの違いを説明し、 ユーザーが最も適切なインストール・シナリオを選択するのに役立つ情報を提供します。すべての製品に インストール・シナリオがあるわけではありません。そのような製品の Classic Express ランチパッド・テンプレートを カスタマイズする場合は、このセクションをナビゲーションから除去できます。インストール・シナリオのページには、 引数付きのカスタム・プログラムの実行のためのコードも用意されています。
  • マイグレーション情報 - マイグレーションのコンテンツでは、通常、 製品の古いバージョンからより新しいバージョンにアップグレードするためにユーザーが行う必要のある、 マイグレーション・パスについて説明します。マイグレーション情報が特定の製品に該当しない場合は、 このコンテンツはナビゲーションから除去してください。マイグレーション情報のページには、PDF の表示およびアプリケーションの実行のためのコードも用意されています。
  • インストール - 製品インストールのページで、製品のインストールを どのように開始するかを指定することができます。 ランチパッドの設定で複数の製品インストールのページを追加すれば、複数の製品をインストールすることができます。 インストール・ページには、製品のインストールを開始するコードが提供されます。
  • 終了 - ナビゲーション・メニューの終了リンクすると、ランチパッドが終了します。
