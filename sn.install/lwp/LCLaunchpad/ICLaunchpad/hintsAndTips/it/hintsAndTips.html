Consigli e suggerimenti per Classic Express Launchpad Template

Classic Express Launchpad Template fornisce un punto di avvio per creare un launchpad di installazione che è possibile utilizzare con prodotti Express.

Questo modello contiene codice che è possibile utilizzare per avviare un programma di installazione del prodotto, visualizzare una documentazione e fornire il contenuto individuato.

Informazioni da conoscere su Classic Express Launchpad Template:

  • Tutti i comandi e i programmi utilizzati sono definiti in launchpadContent/global.properties.
  • Le applicazioni di esempio sono incluse in DISK1 nella pagina Layout dischi. Per aggiungere le applicazioni o i programmi di installazione personalizzati, definire l'applicazione nel file launchpadContent\global.properties (seguendo gli esempi per le applicazioni di esempio). Importare le applicazioni sul disco appropriato del progetto utilizzando i controlli rilevati nella scheda Layout dischi.
  • Tutti i file HTML utilizzano lo stesso foglio di stile (content.css), che risiede nella directory launchpadContent.

Classic Express Launchpad Template contiene le seguenti pagine:

  • Benvenuti - Di solito, la pagina Benvenuti fornisce una panoramica del prodotto ed una descrizione delle funzioni del launchpad. La pagina di benvenuto deve essere sintetica ed offrire le opzioni dell'utente per reperire ulteriori informazioni sul prodotto offrendo un collegamento al sito Web del prodotto visualizzato in una nuova finestra del browser. La pagina di benvenuto è caratterizzata dal codice per visualizzare un sito Web esterno in un browser. Utilizzare la funzione viewPage() per visualizzare le nuove finestre invece di window.open() di JavaScript poiché window.open() appare bloccato dal blocco popup.
  • Informazioni sulla release - La pagina delle informazioni sulla release in genere fornisce informazioni specifiche per una releaseesclusiva di un prodotto. Ad esempio, problematiche di installazione, modifiche dei prerequisiti, funzioni principali o correzioni trattate nella release e altri contenuti di questo tipo rappresentano argomenti appropriati per una pagina di informazioni sulla release. La pagina delle informazioni sulla releaseè caratterizzata dal codice per visualizzare un file PDF ed eseguire un'applicazione.
  • Informazioni sui prerequisiti - Questa pagina in genere fornisce un controllo per l'esecuzione di un programma di verifica dei prerequisiti che determina se nel computer sono presenti prerequisiti di sistema e restituisce un elenco dei prerequisiti non rilevati. La pagina di informazioni sui prerequisiti è caratterizzata dal codice per visualizzare un file PDF ed eseguire un'applicazione.
  • Scenario di installazione - Il contenuto degli scenari di installazione in genere fornisce i diversi approcci per l'installazione di un prodotto che possono essere adottati da un utente. Gli utenti possono scegliere uno specifico scenario di installazione a seconda dell'utilizzo previsto del prodotto. Questo contenuto spiega le differenze dei diversi scenari di installazione che possono essere scelti da un utente e fornisce informazioni sui motivi della scelta di uno scenario rispeto a un altro. Non tutti i prodotti presentano scenari di installazione. Se si personalizza Classic Express Launchpad Template per un prodotto di questo tipo, è possibile rimuovere questa sezione dalla navigazione. La pagina degli scenari di installazione contiene codice per eseguire una programma personalizzato con un argomento.
  • Informazioni sulla migrazione - Il contenuto della migrazione in genere illustra un percorso di migrazione che deve essere utilizzato dall'utente per eseguire l'aggiornamento da una versione precedente del prodotto a una più recente. Se le informazioni sulla migrazione non sono relative a un prodotto specifico, questo contenuto deve essere rimosso dalla navigazione. La pagina delle informazioni di migrazione è caratterizzata dal codice per visualizzare un file PDF ed eseguire un'applicazione.
  • Installa - La pagina di installazione del prodotto fornisce un controllo per avviare l'installazione del prodotto. È possibile aggiungere più pagine di installazione del prodotto a un launchpad di installazione per il supporto di installazioni di più prodotti. La pagina di installazione è caratterizzata dal codice per avviare un'installazione del prodotto.
  • Esci - Il collegamento di chiusura nel menu di navigazione chiude il launchpad.
